# House-Price-Prediction-Using-Lasso-and-Ridge-Regression
Project Overview

This project predicts house prices using the Melbourne Housing Market dataset. The objective is to demonstrate how L1 (Lasso) and L2 (Ridge) Regularization help reduce overfitting and improve model generalization compared to standard Linear Regression.

Dataset:
Melbourne Housing Market Dataset
Source: Kaggle
34,857 records before preprocessing

Technologies Used:
Python
Pandas
NumPy
Scikit-Learn
Matplotlib
Seaborn

Workflow:
Data Cleaning and Missing Value Handling
Feature Selection
One-Hot Encoding of Categorical Variables
Train-Test Split
Linear Regression Model
Lasso (L1) Regression
Ridge (L2) Regression
Performance Comparison

Key Findings:
Linear Regression suffered from overfitting.
Lasso (L1) Regularization improved generalization by shrinking less important coefficients.
Ridge (L2) Regularization reduced coefficient magnitudes and improved prediction performance.
Both regularization techniques significantly improved test accuracy.
